# Architecture Docs

This directory holds the current architecture reference for Mr.Market.

- `market-making-flow.md`: primary market-making runtime flow
- `server/diagrams.md`: Mermaid diagrams (three-layer architecture, execution-layer flow, module map, state machines)
- `server/module-map.md`: module dependency map and business purpose by domain
- `server/business-flows.md`: backend runtime and business flows
- `decisions/`: architecture decision records
